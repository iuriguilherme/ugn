# ugn
Useless general purpose neural network - for learning purposes

## Por que

Eu decidi que preciso estudar redes neurais desde os fundamentos básicos mais uma vez.  

A documentação está escrita em português brasileiro porque escrever isto em inglês é uma burrice que me ensinaram que não tem nenhuma vantagem prática. Estou sempre disposto a ser convencido do contrário.  

## Pra que

Isto está sendo documentado nas plataformas onlines à minha disposição pra convencer burocráticos acadêmicos.  

## Como
